MAD (Mobile App Development) Assignment Portfolio
Welcome to my Android development project collection. This repository contains 4 distinct applications (Tasks) built using Java and Android Studio, covering everything from UI design to hardware sensors and media playback.

📱 Project 1: Currency Converter (X-Change)
A simple and clean tool to convert money between different world currencies.

What it does: Converts values between INR, USD, JPY, and EUR.

Key Feature: Includes a Settings Page with a Dark Mode/Light Mode toggle that changes the look of the entire app.

Skills used: Logic math, Activity switching (Intents), and Theme management.

🎵 Project 2: Media Player
An app designed to handle both local and online entertainment.

Audio: Plays MP3 files stored directly on the phone's disk.

Video: Streams video content from the internet using a URL.

Controls: Full control bar including Play, Pause, Stop, and Restart buttons for both audio and video.

Skills used: MediaPlayer API, VideoView, and handling internet streams.

🧭 Project 3: Sensor Dashboard
This app "talks" to the phone's hardware to see what is happening in the real world.

Accelerometer: Tracks the phone's movement and tilt.

Light Sensor: Detects how bright or dark the room is.

Proximity Sensor: Detects when the phone is held close to an object (like your ear).

Skills used: Hardware SensorManager and real-time data listeners.

📸 Project 4: Photo Manager & Gallery
A complete photo-taking and management app.

Camera: Open the phone's camera, snap a photo, and save it to a specific folder.

Gallery Grid: View all your saved photos in a nice 3-column grid, just like the real Android Gallery.

Image Details: Click any photo to see its "hidden" info: Name, Path, Size, and Date.

Safe Delete: Delete photos you don't want with a "Are you sure?" confirmation pop-up.

Skills used: Camera Intents, FileProvider, RecyclerView (Grids), and File permissions.

🛠️ Tools Used
Language: Java

Software: Android Studio

Testing: Pixel 7 Emulator (API 33+)
